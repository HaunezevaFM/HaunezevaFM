Система контроля версий (VCS):

инструмент, отслеживающий изменения в файлах и позволяющий вернуться к предыдущим версиям.

Git относится к распределенным VCS (DVCS):

Главное отличие от централизованных VCS: каждый разработчик имеет полную копию репозитория, что обеспечивает большую автономность и устойчивость к сбоям.

Преимущества Git:

распределенность, ветвление, скорость, гибкость для параллельной разработки и управления изменениями.

Git против GitHub:

Git — локальная система контроля версий; GitHub — веб-платформа для размещения Git-репозиториев и совместной работы.
